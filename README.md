BookSky – Personal Book Library
BookSky is a simple web application that allows users to manage their personal book collection. Users can add new books, view existing books, and delete books as needed. The project is built using HTML, CSS, and JavaScript.

Features
Add Books: Add a book with its title, author, and description using a popup form.

Delete Books: Remove books from the library easily.

Responsive UI: Modern and colorful design using Poppins font and styled buttons.

Popup Form: Intuitive popup overlay for adding new books.

Technologies Used
HTML5 – Structure of the web pages

CSS3 – Styling and layout

JavaScript – Adding interactivity and handling dynamic content

Google Fonts – Poppins font for modern typography



Usage
Click the + button at the bottom right corner to open the "Add Book" popup.

Fill in the Book Title, Author, and Description fields.

Click Add to save the book.

To remove a book, click the delete button on the book card.
